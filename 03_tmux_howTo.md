# [tmux]

## shell 화면 분할 툴


##  [설치]
```$ sudo yum install tmux```


## [명령어]


#### 세션 생성
- 0 부터 숫자를 쌓아 나감

    ```$ tmux```


#### 세션 리스트 업

```$ tmux ls```


#### 특정 세션으로 이동

```$ tmux at -t [세션 번호 또는 이름]```



#### 가로 분할

- (ctrl + b) %


####  세로 분할

- (ctrl + b) "



#### 분할 내 이동

- (ctrl + b) 방향키



#### 세션 나가기

- (ctrl + b) d



#### pane 닫기

- exit
